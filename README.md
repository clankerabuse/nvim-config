# 💤 LazyVim

A customized LazyVim config with a few tweaks:

- **Tabs over spaces** (4-wide)
- **vscode.nvim** colorscheme
- **avante.nvim** with opencode provider
- **supermaven-nvim** for inline completions
- Custom cursor styling

Refer to the [documentation](https://lazyvim.github.io/installation) to get started.
